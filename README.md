
# TikTok Signup UI

A simple and responsive TikTok-inspired signup interface built using **HTML and CSS**.

---

## 📌 Project Overview

This project is a front-end UI clone of a TikTok-style signup page. It focuses on clean layout, responsive design, and modern UI principles.

---

## 🚀 Features

* Responsive design for all screen sizes
* Clean and modern UI layout
* Hover effects on buttons and links
* Flexbox-based layout structure
* Mobile-friendly interface

---

## 🛠️ Built With

* HTML5
* CSS3
* Flexbox
* Google Fonts (Poppins)
* Ionicons (ionic icons)

---

## 📱 Responsive Design

The layout is optimized for:

* Mobile devices
* Tablets
* Desktop screens

It uses `max-width` and flexible units instead of fixed widths for better adaptability.
